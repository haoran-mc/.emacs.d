# vertico

垂直 minibuffer 补全，原生的 emacs 补全例如： evil-1 | evil-2 | evil-3 | ...，横着补全排版更乱。

# orderless

支持 minibuffer 模糊补全。

# marginalia

显示命令的解释、变量的值、快捷键绑定、文件的大小、文件的权限属性、buffer major mode等。

# embark

M-o 增强 minibuffer。

C-x C-h 不需要再记快捷键了，通过模糊查找就可以执行有快捷键的命令。

其实这个反而用得少了。

# consult

替换了原本的 emacs 的一些功能，比如 recentf、mark、bookmark 等。

C-s 替换了 ivy 的 swiper 功能，更强大的搜索能力。

还有个非常强大的 consult-imenu！
